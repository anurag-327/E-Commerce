# E-Commerce

## E-commerce site built using react js
- Backend-Express js
- Frontend-React Js
- Database-MongoDB

### Steps run code on your machine
   - Clone repo on your 
   - Start Backend Server
         **cd api**
         **npm i**
         **npm run start**
   - start Frontend vite server
         **cd client**
         **npm run dev**
   - You are good to go
